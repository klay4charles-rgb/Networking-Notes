# Networking Devices

## Main Idea

Networking devices are responsible for moving, securing, managing, and optimizing network traffic. Each device has a specific role within a network and helps ensure reliable communication between systems.

## Router

A router connects different networks together and forwards traffic between them.

Functions:

* Connects multiple networks
* Routes packets between networks
* Determines the best path for traffic
* Often serves as the gateway to the internet

Modern routers may also include switching and firewall capabilities.

## Switch

A switch connects devices within the same local area network (LAN).

Functions:

* Uses MAC addresses to forward traffic
* Reduces unnecessary network traffic
* Improves network efficiency
* Provides communication between local devices

Switches primarily operate at Layer 2 of the OSI model.

## Firewall

A firewall controls which traffic is allowed into or out of a network.

Functions:

* Filters network traffic
* Blocks unauthorized access
* Enforces security policies
* Protects internal systems from external threats

Firewalls can be hardware-based, software-based, or both.

## IDS and IPS

### Intrusion Detection System (IDS)

An IDS monitors network traffic and alerts administrators when suspicious activity is detected.

Functions:

* Traffic monitoring
* Threat detection
* Security alert generation

An IDS does not automatically block traffic.

### Intrusion Prevention System (IPS)

An IPS performs the same monitoring functions as an IDS but can automatically take action.

Functions:

* Detect malicious activity
* Block harmful traffic
* Prevent attacks in real time

## Load Balancer

A load balancer distributes network traffic across multiple servers.

Benefits:

* Improved performance
* Better availability
* Increased reliability
* Reduced server overload

Load balancing helps prevent a single server from becoming a bottleneck.

## Proxy Server

A proxy acts as an intermediary between a client and another server.

Functions:

* Content filtering
* Traffic monitoring
* Privacy enhancement
* Web caching
* Security enforcement

Organizations commonly use proxies to control internet access and improve performance.

## NAS

NAS stands for Network Attached Storage.

Functions:

* Centralized file storage
* File sharing across a network
* Easy access from multiple devices

NAS is commonly used for file storage and backups.

## SAN

SAN stands for Storage Area Network.

Functions:

* High-speed storage access
* Enterprise-level storage solutions
* Block-level storage

SAN environments are commonly found in data centers and large organizations.

## Wireless Access Point

A Wireless Access Point (WAP) allows wireless devices to connect to a wired network.

Functions:

* Provides Wi-Fi connectivity
* Extends network access
* Connects wireless devices to the LAN

Access points are commonly used in homes, offices, schools, and public spaces.

## Key Takeaway

Networking devices each serve a specialized purpose. Routers connect networks, switches connect local devices, firewalls secure traffic, IDS/IPS detect threats, load balancers distribute traffic, proxies manage access, NAS and SAN provide storage solutions, and access points enable wireless connectivity.
