# Networking Functions

## Main Idea

Modern networks do more than simply connect devices. Networking functions help improve performance, security, reliability, and communication between systems.

These functions ensure that data can be delivered efficiently and securely across local and global networks.

## Access to Data

One of the primary purposes of networking is to provide access to information and resources.

Benefits include:

* File sharing
* Application access
* Resource sharing
* Centralized data management

Networks allow users to access information regardless of where the data is physically stored.

## Remote Access

Remote access allows users to connect to systems and resources from different locations.

Benefits include:

* Work from home capabilities
* Remote administration
* Access to company resources from anywhere
* Improved flexibility

Remote access has become an essential part of modern networking.

## Traffic Management

Networks must manage traffic efficiently to prevent congestion and maintain performance.

Goals include:

* Efficient data delivery
* Reduced network congestion
* Improved user experience
* Better utilization of network resources

Traffic management helps ensure that important applications continue to function properly even during periods of heavy usage.

## Protocol Support

Protocols define the rules used for communication between devices.

Examples include:

* TCP
* UDP
* IP
* HTTP
* HTTPS
* DNS

Without protocols, devices would not be able to communicate in a standardized way.

## Content Delivery Network (CDN)

A CDN is a distributed network of servers that delivers content from locations closer to users.

Benefits include:

* Faster content delivery
* Reduced latency
* Improved availability
* Reduced load on origin servers

CDNs are commonly used by websites, streaming services, and cloud providers.

## Virtual Private Network (VPN)

A VPN creates an encrypted connection between a user and a remote network.

Benefits include:

* Improved privacy
* Secure remote access
* Data encryption
* Protection when using public networks

VPNs are commonly used by organizations and remote workers.

## Quality of Service (QoS)

Quality of Service (QoS) prioritizes specific types of network traffic.

Examples:

* Voice calls
* Video conferencing
* Real-time applications

Benefits include:

* Reduced latency
* Reduced jitter
* Improved performance for critical applications

QoS helps ensure that important traffic receives priority over less critical traffic.

## Time To Live (TTL)

TTL is a value stored within an IP packet that limits how long it can travel across a network.

How it works:

* Each router decreases the TTL value by 1
* When TTL reaches 0, the packet is discarded
* Prevents packets from traveling indefinitely

TTL helps protect networks from routing problems and infinite loops.

## Routing Loops

A routing loop occurs when routers continuously forward packets in a circular path.

Problems caused by routing loops:

* Network congestion
* Wasted bandwidth
* Increased latency
* Packet loss

TTL helps prevent routing loops from consuming network resources indefinitely.

## Domain Name System (DNS)

DNS translates human-readable domain names into IP addresses.

Example:

* google.com → IP address

Benefits include:

* Easier navigation
* Simplified network communication
* User-friendly internet access

DNS acts as the internet's phonebook by helping devices locate resources using names instead of numerical addresses.

## Key Takeaway

Networking functions provide the services that make modern communication possible. Features such as remote access, CDN, VPN, QoS, TTL, routing protection, and DNS help improve network performance, security, and reliability.
